# 📌 About the Project

House Price Prediction is a Machine Learning project developed using **Python** and **Scikit-learn**. The model predicts the median house value based on features from the California Housing Dataset.

This project demonstrates a complete Machine Learning workflow, including:

- 📥 Data Loading
- 🧹 Data Preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 🤖 Linear Regression Model
- 📈 Model Evaluation
- 📉 Prediction Analysis

# 📊 Model Performance

| Metric | Score |
|--------|------:|
| Mean Absolute Error (MAE) | **0.5332** |
| Mean Squared Error (MSE) | **0.5559** |
| Root Mean Squared Error (RMSE) | **0.7456** |
| R² Score | **0.5758** |

---

# 📈 Result Summary

✔ Successfully trained a **Linear Regression Model**

✔ Achieved an **R² Score of 0.5758**

✔ Model explains approximately **57.6%** of the variation in house prices.

✔ Performance indicates a good baseline model that can be further improved using advanced regression algorithms.

# 🚀 Installation

Follow these steps to run the project on your local machine.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/aashurajput22/House-Price-Prediction.git
```

---

### 2️⃣ Navigate to the Project Folder

```bash
cd House-Price-Prediction
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Launch the Notebook

Open the notebook using Jupyter Notebook or Google Colab.

```bash
jupyter notebook
```

or upload the notebook to **Google Colab** and run all cells.
